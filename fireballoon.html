Fire Balloon Study Assistant

  • Front-end: Express (Node.js), Socket.IO, Bootstrap
  • Back-end: Dialogflow, Flask (Python), Gunicorn, mLab (MongoDB)
  • Hosting: Heroku

This project was done as part of a university course. Thus, my contributions on GitHub are under my school account, z5025212. The Fire Balloon Study Assistant is a conversational agent (chat bot) that can be used to study for university courses. Given a text resource, Fire Balloon splits the text into chunks and stores these chunks in a database. When the user asks a question, Fire Balloon analyses the query for keywords and gives each chunk a score based on its relevance. The most relevant chunks are returned to the user. For this example, we have pre-loaded the lecture slides from the "neural networks and deep learning", "Artificial Intelligence", and "Software Engineering Fundamentals" courses.

Fire Balloon is also able to learn from user feedback by allowing users to select the most relevant response. This will increase its "relevance score" slightly when matched with the same keywords.

Example usage: The "restart" command is used to reset the database and "untrain" what the AI has learnt from user feedback. It also pings the backend Heroku server which takes a couple of seconds to boot up.

The orange buttons can be clicked to open the PDF at the page the information came from.
